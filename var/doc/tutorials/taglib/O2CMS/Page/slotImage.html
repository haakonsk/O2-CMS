Display/edit an image. Default image is provided by the "field" attribute, which is the image id field. If an image has been dropped on the slotImage, this image takes precedence. Either "width" or "height" must be specified. If only "width" or only "height" is given, the original width/height ratio is usually maintained. field - name of the object field that returns the id of the image. width - width of the image, in pixels. At least one of width, height must be given. height - height of the image, in pixels. At least one of width, height must be given.
