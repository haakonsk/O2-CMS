In order to publish content with drag and drop in O2, we use "slots". A slot is a tag in the template which in the editor is displayed as an editable box, while on the web can display anything. "Anything" depends on two things: a template and possibly a data object. If you drop a template (grid) on a slot, the template's content will appear in the box - no magic here. A grid is like an include file, and may contain everything from a static page header to new slots with beautiful layout. If you drop a data object on a slot, O2 will display the object rendered through an "object template". An object template is a template written to display information from a data class, and which is registered for use with this class. So if we drop an article on a slot, O2 will search for an object template that is registered for use with O2CMS::Obj::Article.
