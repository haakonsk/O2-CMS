This is an element used mainly in the backend to drag from the tree menu on the left side over to a dragList in the main frame. .dragList { border : 1px dotted black; } .dragList:before { content : ""; } id class items - an array reference with the objects to appear in the dragList when the page is loaded. allowClass [*] - a comma-separated list of allowed classes. Objects of other classes can't be dropped on the dragList. The default (*) allows all objects. replace [yes] - {yes, no} yes: replace item under mouse on drop (instead of adding). reorganize [yes] - {yes, no} yes: allow list to be reorganized by dragging items to a new position. unique [yes] - {yes, no} yes: do not add item if it already exists in list. removeOnDragEnd [yes] - {yes, no} yes: remove item from list when dragged to another list. content - tags.
